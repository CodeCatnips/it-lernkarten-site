# IT-Lernkarten App

Ein einfaches Webtool zur Vorbereitung auf die IHK-Prüfung für Fachinformatiker.

##   Funktionen

- Zufällige Lernkarten (Frage/Antwort)
- Themen: Betriebssysteme, Netzwerke, Sicherheit
- "Umdrehen"-Button und "Nächste Karte"

##   Technologien

- HTML, CSS, JavaScript
- Kein Backend, keine Datenbank — funktioniert offline
